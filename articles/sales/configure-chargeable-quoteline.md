# <a name="configure-the-chargeable-components-of-a-project-based-quote-line"></a><span data-ttu-id="b798f-103">Konfigurisanje naplativim komponentama stavke ponude zasnovane na projektu</span><span class="sxs-lookup"><span data-stu-id="b798f-103">Configure the chargeable components of a project-based quote line</span></span>

<span data-ttu-id="b798f-106">Uključene komponente podležu načinu naplate, podeli između klijenata, ograničenjima koja nije dozvoljeno prekoračiti i podešavanjima učestalosti fakturisanja definisanim na stavci ponude zasnovanoj na projektu.</span><span class="sxs-lookup"><span data-stu-id="b798f-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based quote line.</span></span>

   - <span data-ttu-id="b798f-110">Nenaplativo</span><span class="sxs-lookup"><span data-stu-id="b798f-110">Non-chargeable</span></span>

<span data-ttu-id="b798f-114">Naplativost koja je definisana na kategorijama transakcija za stavku ponude za projekat primenjuje se samo na klasu transakcije **Trošak**.</span><span class="sxs-lookup"><span data-stu-id="b798f-114">Chargeability that is defined on transaction categories for a project quote line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="b798f-115">Ako stavka ponude za projekat ima **Uključi trošak** = **Ne**, kartica **Naplative kategorije** nije dostupna.</span><span class="sxs-lookup"><span data-stu-id="b798f-115">If a project quote line has **Include Expenses** = **No**, the **Chargeable Categories** tab isn't available.</span></span>
